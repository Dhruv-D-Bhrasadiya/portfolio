# Portfolio Website

This is a portfolio website generated from a resume.

## How to view locally

1. Open the `index.html` file in your web browser.

## How to deploy to GitHub Pages

1. Make sure you have a GitHub account.
2. Create a new repository on GitHub.
3. Push the files in this directory to the new repository.
4. In the repository's settings, go to the "Pages" section.
5. Select the `main` branch as the source and click "Save".
6. Your portfolio will be available at `https://<your-username>.github.io/<repository-name>`.

## Notes

* The LinkedIn and GitHub links in the "Contact" section are placeholders. Please update them with your actual links.
* The project links are also placeholders. Please update them with your actual links.
